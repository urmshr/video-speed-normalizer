# Video Speed Normalizer

YouTube で指定した条件に当てはまる動画を再生したときに再生速度を自動で 1.0 倍 に戻す Chrome 拡張機能。

ミュージックビデオや特定のジャンルの動画は常に 1.0 倍で再生したい場合に便利です。

## 機能

### 設定したキーワードが含まれるなら 1.0 倍で再生する

設定したキーワードがタイトルまたはチャンネル名に含まれる場合、その動画を 1.0 倍で再生します。  
※ タイトルだけでなくチャンネル名も判定に用いるかは設定で切り替え可能（デフォルトではチャンネル名も対象）

デフォルトでは以下のようなキーワードが設定されています。キーワードは自由に追加・削除できます。  
`MV`, `music`, `Choreography`, `踊ってみた`, `コント`, `漫才` など

### タイトルが特定の形式なら 1.0 倍で再生する

タイトルが以下の形式に当てはまる場合、その動画を 1.0 倍で再生します。

- `aaa「bbb」`
- `aaa『bbb』`
- `aaa - bbb`
  - ダッシュは`-−‐‒–—－ーｰ`のいずれか
- `aaa / bbb`
- `aaa ／ bbb`

※ この機能は設定で有効・無効を切り替え可能（デフォルトでは有効）

### 除外キーワードを設定する

除外キーワードとして指定したキーワードがタイトルまたはチャンネル名に含まれる場合、再生速度を変更しません。

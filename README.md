# Video Speed Normalizer

YouTube で設定した条件に当てはまる動画を再生したときに再生速度を自動で 1.0 倍 に戻す Chrome 拡張機能

ミュージックビデオや特定のジャンルの動画は常に 1.0 倍で再生したい場合に便利です。

## 機能

設定した条件に一致した場合に再生速度を 1.0 倍にします。

### キーワードによる判定 🗝️

タイトルやチャンネル名に設定したキーワードが含まれている場合、その動画を 1.0 倍で再生します。

デフォルトでは以下のようなキーワードが設定されています。  
`MV`, `music`, `choreography`, `踊ってみた`, `コント`, `漫才` など

### タイトル形式による判定 🎬

動画のタイトルが以下の形式に当てはまる場合、その動画を 1.0 倍で再生します。

- `aaa「bbb」`
- `aaa『bbb』`
- `aaa - bbb`（全角・半角など複数のハイフン・ダッシュ記号に対応）
- `aaa / bbb`（全角・半角スラッシュに対応）

### 公式アーティストバッジによる判定 🎵

チャンネル名の横に公式アーティストバッジが表示されている場合、その動画を 1.0 倍で再生します。

### 概要欄の「音楽」セクションによる判定 💿

動画の概要欄に「音楽」セクションがある場合、その動画を 1.0 倍で再生します。

### 除外キーワードの設定 🚫

設定した除外キーワードが含まれる場合は、他の条件に当てはまったとしても再生速度を変更しません。

### 設定 ⚙️

拡張機能のポップアップから各種設定を変更できます。

- キーワードの追加・削除
- 除外キーワードの追加・削除
- チャンネル名でのキーワード判定の有効・無効化
- タイトル形式による判定の有効・無効化
- 公式アーティストバッジによる判定の有効・無効化
- 概要欄の「音楽」セクションによる判定の有効・無効化
